Objetivo

Desenvolver uma ferramenta "server-side" de pesquisa para localizar tópicos e posts na cópia estática do Fórum Clube Cético a partir de palavras chaves.

Etapa 1

Criar um banco de dados com os dados brutos coletados das páginas do acervo.

As páginas contém dados de: usuários, posts, enquetes, seções, subseções e tópicos. Estas informações já foram coletadas em um projeto anterior e estão disponíveis em um conjunto de arquivos json.

Nesta etapa será definida a estrutura desse banco de dados e também será desenvolvido o código em linguagem Java para processar os arquivos json e inserir os dados em tabelas do BD. (O SGBD utilizado será o MySQL)
